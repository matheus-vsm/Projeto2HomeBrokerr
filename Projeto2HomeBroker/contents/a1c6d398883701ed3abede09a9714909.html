UMLUseCase
Projeto 2 - Home Broker :: Diagrama de Casos de Uso :: Modificação

Description

Modificação Ordem - Caso de Uso responsável pela alteração de informações da ordem de venda ou compra do usuario na plataforma.

Fluxo Principal:

1 - Usuario escolhe a ordem que deseja alterar.

2 - Usuario modifica os quesitos que deseja.

3 - Modificações aceitas e ordem atualizada no sistema.

Fluxo Alternativo:

2 - Erro ou mudança não permitida.

3 - Tente novamente mais tarde ou peça ajuda ao suporte.
