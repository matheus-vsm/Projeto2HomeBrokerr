Description

Consultar Historico de Operações - Caso de Uso responsável pela consulta do historico das operacoes do usuario na plataforma.

Fluxo Principal:

1 - Usuario clica para visualizar o historico

2 - Retorna historico das operacoes.

Fluxo Alternativo:

2 - Erro ao acessar historico das operações.

3 - Tente novamente mais tarde ou busque ajuda no suporte.
