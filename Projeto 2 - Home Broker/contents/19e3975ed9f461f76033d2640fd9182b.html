Cadastro do Usuario

UMLUseCase
Projeto 2 - Home Broker :: Diagrama de Casos de Uso :: Cadastro do Usuario

Description

Cadastro do Usuario - Caso de Uso responsável por cadastrar o usuario na plataforma.

Fluxo Principal:

1 - Usuario preenche as informações necessarias para criação da sua conta.

2 - Cadastro concluido e nova conta criada.

Fluxo Alternativo:

2 - Email invalido ou usuario ja cadastrado

3 - Retorna para pagina inicial
